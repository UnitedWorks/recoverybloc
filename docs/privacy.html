Privacy Statement

Recovery Bloc provides software that transforms computing power into cryptocurrency, which is then transformed into funds and given to Puerto Rican community leaders. 100% of the resources generated using the software are distributed to those leaders.

INFORMATION WE HAVE

We do not collect information of any participants. The Recovery Bloc software does not collect or transmit any identifying information of the user's computer other than the IP address, which is used on a temporary basis by the mining pool (XMRpool.net) to prevent abuse of the network.

INFORMATION WE SHARE

Because we do not collect any information about our participants, we have nothing to share, even if required by law.

OUR SECURITY

We use HTTPS, an encrypted protocol that establishes communication between the user's computer and our web server. Our code is open source so that it can be vetted by our community. The Recovery Bloc software mines Monero, which is an anonymous and decentralized cryptocurrency.
